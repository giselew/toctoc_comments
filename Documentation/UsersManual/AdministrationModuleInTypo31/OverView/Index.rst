Overview
^^^^^^^^

The overview panel is first shown when you open the backend module.
It contains summaries about comments, ratings, users and reports, you can maintain blacklists and optimize database tables directly from the overview panel





.. figure:: /Images/backendoverview.jpg

The overview can be minimized and refreshed with the buttons on the right top.
