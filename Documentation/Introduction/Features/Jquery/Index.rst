jQuery
^^^^^^

Uses jQuery 1.8 or newer

jQuery plugins used
"""""""""""""""""""

Inputfields with watermark using jQuery Watermark

Textarea auto grow is implemented with mofified jQuery Elastic

Tooltips and Image presentation with mofified jQuery Tools

Sharing component sharrre is made implemented with jQuery sharrre and AddThis

Dialog popups use mofified jQuery Simple modal (Simpletxtcmoddlg)

Javascript library flowplayer is used for HTML5 video
