Features
--------

.. index::
	single: Features; Overview

The extension offers almost all of the possible features that are needed for commenting,
reviews, rating, sharing, login/logout and sign up. 

Many of the features have been brought in by users of **toctoc_comments.** 

The feature list is long and it's important to have a complete overview for taking the right
decisions when you start use the extension.
