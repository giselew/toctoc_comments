topSharings
-----------

All topSharings options are valid per webpage
(W)

=======================  ==========  ====================================================  ========
**Property:**            Data type:  Description:                                          Default:
=======================  ==========  ====================================================  ========
topSharingsMode          options     What to show?: Show top sharings for each sharer (0)  0
                                     or each page (1)                                     
                                     (V 8.0.0)
-----------------------  ----------  ----------------------------------------------------  --------
**Options setup-only**
-----------------------  ----------  ----------------------------------------------------  --------
topSharingsLookBackDays  int+        Show top sharing as they were this number of          0
                                     days ago
                                     (V 8.0.0)
